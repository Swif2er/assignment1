# assignment1

### Instructions:

1. Clone repository:
```
git clone https://github.com/Swif2er/assignment1.git
```
2. Start servers:
```
docker-compose up -d
```
3. Access servers via load balancer:
```
http://localhost:8080
```

There are 2 servers running and accessible directly on localhost ports 5001 & 5002
4. Stop servers:
```
docker-compose down
```
